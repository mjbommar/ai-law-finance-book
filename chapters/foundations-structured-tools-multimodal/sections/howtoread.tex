\section*{How to Read This Chapter}
\addcontentsline{toc}{section}{How to Read This Chapter}

If you need trustworthy, auditable outputs and tool calls, focus on Sections~\ref{sec:llmC-structured} and \ref{sec:llmC-tools}. Multimodal inputs are covered in the next chapter.

\begin{keybox}[title={Key Objectives}]
\begin{itemize}
  \item Design schemas and validators to get reliable JSON/CSV/XML.
  \item Call tools safely with pre/postconditions and governance metadata.
  \item Handle common multimodal inputs (PDFs, tables, charts, audio).
\end{itemize}
\end{keybox}
