\section{Telemetry, Monitoring, and Evidence Pipelines}
\label{sec:llmE-telemetry}

% Outline (comments): latencies, error classes, escalation counts; dashboards; tamper-evident logs
\begin{keybox}[title={Operational KPIs (Examples)}]
\begin{itemize}
  \item Latency percentiles (P50/P95), timeout/error rates, retry/backoff counts.
  \item Escalation/abstention rates; human-in-the-loop turnaround.
  \item Citation fidelity pass rate; schema validation failure rate.
\end{itemize}
\end{keybox}

\paragraph{Evidence Pipelines.} Hash and timestamp logs; store prompts, configurations, and outputs with correlation IDs. Export slices for DPIA/TRA and Part~III audits.
