\section*{How to Read This Chapter}
\addcontentsline{toc}{section}{How to Read This Chapter}

If you build prompts today but struggle with multi-turn workflows or reliability, focus on Sections~\ref{sec:llmB-convo} and \ref{sec:llmB-reason}.

\begin{keybox}[title={Key Objectives}]
\begin{itemize}
  \item Design conversations with clear roles and state.
  \item Apply reasoning strategies that fit task risk and constraints.
  \item Avoid over-engineering when a simpler pattern suffices.
\end{itemize}
\end{keybox}
