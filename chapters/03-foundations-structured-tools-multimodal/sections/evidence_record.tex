\section{Canonical Evidence Record}
\label{sec:llmC-evidence}

We standardize how to capture evidence for any claim the system makes.

\begin{definitionbox}[title={Evidence Record (Fields)}]
\begin{itemize}
  \item \textbf{source}: stable URL/DOI or repository identifier
  \item \textbf{locator}: page/paragraph/figure/table identifiers
  \item \textbf{quote}: exact quoted text (or figure/table description)
  \item \textbf{date}: publication/effective date; include timezone
  \item \textbf{jurisdiction}: e.g., US/MI; sector flags (HIPAA/GLBA, etc.)
  \item \textbf{hash}: content hash (e.g., SHA-256) of the cited artifact
  \item \textbf{model}: model name/version; sampling params; seeds
  \item \textbf{correlationId}: run/request ID linking logs and actions
\end{itemize}
\end{definitionbox}

\end{highlightbox}
