\section{JSON-LD for Web Interop}
\label{sec:kg-foundations-jsonld}

JSON-LD 1.1 enables publishing and consuming RDF via familiar JSON, using contexts to map keys to IRIs and optional framing for client-friendly shapes. It is an on-ramp for applications and APIs. \autocite{w3c-jsonld11}

\begin{keybox}[title={Publishing Tips}]
\begin{itemize}
  \item Serve machine-readable JSON-LD alongside human pages.
  \item Version contexts and document them; avoid breaking changes.
  \item Reuse identifiers from authoritative registries (LEI, CIK, ELI/ECLI).
\end{itemize}
\end{keybox}
