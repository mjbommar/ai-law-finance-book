\section*{How to Read This Chapter}
\addcontentsline{toc}{section}{How to Read This Chapter}

This chapter turns prompting into an engineering discipline: specify, measure, and improve.

\begin{keybox}[title={Key Objectives}]
\begin{itemize}
  \item Write prompts as unambiguous specifications with success criteria.
  \item Build small, repeatable evaluation sets; log what matters.
  \item Use meta-prompting and versioning to improve safely.
\end{itemize}
\end{keybox}
